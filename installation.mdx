## Prerequisites

Before installing effect-orpc, ensure you have the following:

- **Node.js** 18 or later
- **TypeScript** 5.0 or later
- Basic knowledge of [Effect](https://effect.website/) and [oRPC](https://orpc.dev/)

## Install effect-orpc

You can install effect-orpc using your preferred package manager:

<CodeGroup>

```bash npm
npm install effect-orpc
```

```bash pnpm
pnpm add effect-orpc
```

```bash yarn
yarn add effect-orpc
```

```bash bun
bun add effect-orpc
```

</CodeGroup>

## Install peer dependencies

effect-orpc requires several peer dependencies to work correctly. Install them alongside effect-orpc:

<CodeGroup>

```bash npm
npm install effect @orpc/server @orpc/client @orpc/contract @orpc/shared
```

```bash pnpm
pnpm add effect @orpc/server @orpc/client @orpc/contract @orpc/shared
```

```bash yarn
yarn add effect @orpc/server @orpc/client @orpc/contract @orpc/shared
```

```bash bun
bun add effect @orpc/server @orpc/client @orpc/contract @orpc/shared
```

</CodeGroup>

<Note>
  The minimum required versions are:
  - `effect` >= 3.18.0
  - `@orpc/server`, `@orpc/client`, `@orpc/contract`, `@orpc/shared` >= 1.13.0
</Note>

## Verify installation

After installation, verify that effect-orpc is installed correctly by creating a simple test file:

```typescript test.ts
import { Effect, ManagedRuntime, Layer } from "effect";
import { makeEffectORPC } from "effect-orpc";

const runtime = ManagedRuntime.make(Layer.empty);
const effectOs = makeEffectORPC(runtime);

const hello = effectOs.effect(function* () {
  return yield* Effect.succeed("Hello from effect-orpc!");
});

console.log("effect-orpc installed successfully!");
```

Run the file to confirm:

```bash
npx tsx test.ts
```

If you see "effect-orpc installed successfully!", you're ready to build Effect-native oRPC procedures.
